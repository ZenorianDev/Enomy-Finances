<h1>Enomy-Finances</h1>

<p>
Enomy-Finances is a web-based system developed by Phonyt Digital Solutions (PDS) that helps
staff and clients manage currency conversion, savings, and investment planning.
</p>

<img src="src/main/resources/static/images/enomy-finance.gif" alt="enomyfinances" width="600">

<br>

<h2>Project Description</h2>

<p>
The system provides tools for real-time currency exchange, personalized financial quotes,
secure data storage, and a user-friendly interface. Clients can plan investments, track savings,
and make informed financial decisions efficiently.
</p>

<br>

<h2>Key Features</h2>

<ul>
    <li>Real-time currency exchange rates</li>
    <li>Currency conversion tools</li>
    <li>Savings and investment planning tools</li>
    <li>Personalized financial quotes</li>
    <li>Secure storage of user and financial data</li>
    <li>User-friendly web interface</li>
</ul>
